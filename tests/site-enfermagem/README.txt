SITE ESTATICO - CUIDAR+ ENFERMAGEM

Esta pasta esta pronta para publicar gratis no Netlify.

Arquivos principais:
- index.html
- assets/videos/cuidados-enfermagem.mp4

Como publicar:
1. Entre em https://www.netlify.com/
2. Crie uma conta ou faca login.
3. Clique em Add new site.
4. Escolha Deploy manually.
5. Arraste a pasta site-enfermagem para a tela do Netlify.
6. Espere terminar.
7. O Netlify vai gerar um link terminado em .netlify.app.

Esse link pode ser enviado para outra pessoa abrir no celular sem aviso do ngrok.

Importante:
- Sempre que editar o index.html, envie a pasta novamente no Netlify.
- Nao envie a pasta do Laravel inteira. Envie somente esta pasta: site-enfermagem.
